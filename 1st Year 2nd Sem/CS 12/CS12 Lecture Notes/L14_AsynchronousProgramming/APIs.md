## 1. Application Programming Interface (API)
: defines how a program interacts with another

e.g.
 - ABCs and Protocols : are APIs in python such that it defines a template/rules from which classes must follow these to be a subtype of it and to interact with other objects
 - Web API JSON data : it defines a set of data in a JSON format, such that programs must interpret JSON to use the data online
